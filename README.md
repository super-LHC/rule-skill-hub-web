# RuleSkill Hub Web Edition
## Introduction
RuleSkill Hub Web Edition is a simple display and drainage platform, built to support the landing and promotion of Harness AI Coding. It shows the core functions of the RuleSkill Hub Local Edition, highlights the importance of rules (Rule) and skills (Skill) for Harness, and guides users to download the local edition to assist Harness Coding.

## Core Functions
- Display: Show the core functions and application scenarios of the RuleSkill Hub Local Edition, focusing on the support role of Rule and Skill for Harness AI Coding.
- Drainage: Provide download links for the local edition and links to the GitHub repository of the local edition, guiding users to use the local edition to improve Harness Coding efficiency.
- Simple Interaction: Provide a simple message board for users to leave feedback and suggestions related to Harness Coding and the tool.

## Deployment
- Deployed on a 2h4g server, using a paid domain name for easy access.
- It is a static page + simple backend, with simple operation and stable operation, focusing on display and drainage.

## Key Value for Harness AI Coding
As a display and drainage platform for Harness AI Coding, it helps spread the value of Rule and Skill for Harness landing, guides more developers to use the local edition to manage rules and skills, and further promotes the efficient landing of Harness AI Coding.
